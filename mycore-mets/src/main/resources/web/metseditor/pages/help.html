Allgemeines

Mit Hilfe dieses Mets Editors ist es möglich, DFG-Viewer-konforme Metsdokumente zu erzeugen. Nachdem durch Klick auf die Speichern-Schaltfläche ein Metsdokument auf dem Server erzeugt wurde, erscheint unmittelbar danach in der Metadatenansicht ein Link zum DFG-Viewer. Dort können Sie dann die Struktur überprüfen.

Strukturen erstellen und verwalten

Zuerst sollten Sie den Dokumenttyp festlegen. Dazu klicken Sie bitte auf folgende Schaltfläche . Es erscheint ein Dialog. Hier können Sie den Dokumenttyp einfach auswählen und zusätzlich den Titel eingeben, der im DFG-Viewer über der Navigation angezeigt werden soll. Klicken Sie auf Speichern um die eingegebenen Informationen zu übernehmen.

Die weitere Strukturierung des Dokumentes kann nun wie folgt vorgenommen werden. Klicken Sie auf das Wurzelelement des angezeigten Baumes und anschließend auf die folgende Schaltfläche . Es öffnet sich ein Dialog mit dessen Hilfe weitere Einstellungen vorgenommen werden können. Zunächst sollten Sie den Typ der Struktureinheit festlegen, bspw. "Kapitel" danach müssen Sie noch ein Label vergeben. Dafür geben Sie einfach den entsprechenden Text in das dafür vorgesehene Textfeld ein, bspw. "Kapitel 1.5". Die Informationen lassen sich nur übernehmen, wenn ein gültiger Strukturtyp ausgewählt und ein Label angegeben wurde. Klicken Sie auf "Hinzufügen" um die Strukturinformationen zu übernehmen. Mit dem gleichen Vorgehen, können Sie das soeben angelegte Strukturelement weiterstrukturieren. Dafür wählen Sie das Element in der Baumansicht aus und klicken erneut auf . Wollen Sie Änderungen an einen Element vornehmen, so führen Sie entweder einen Doppelklick auf das entsprechende Element aus oder klicken auf nachdem Sie ein Strukturelement ausgewählt haben.

Sie können nun die Seiten auswählen, die Sie in das neue Strukturelement verschieben wollen. Sie können sowohl Einzelseiten verschieben als auch mehrere, mittels Strg- oder Shifttaste, markierte Seiten.

Paginierung/Foliierung

Der Metseditor untersützt auch die Paginierung und Foliierung. Sie müssen wie folgt vorgehen. Wählen Sie in der Baumansicht die Seite mit der Sie die beginnen wollen danach halten Sie die "STRG"/"CTRL"-Taste gedrückt und wählen die Seite bei der die Paginierung/Foliierung enden soll. Nun können Sie auf klicken. Es erscheint ein Dialog mit dem nun die konkreten Einstellungen vorgenommen werden können. Sie müssen jetzt die Art der Paginierung/Foliierung festlegen. Folgende stehen zur Auswahl:
  • Blattnummerierung (mit verso beginnend)
  • Blattnummerierung (mit recto beginnend)
  • römisch (Großbuchstaben)
  • römisch (Kleinbuchstaben)
  • arabisch
  • Umschlag
Geben Sie die Zahl ein mit der bei der Paginierung/Foliierung begonnen werden soll und haken Sie die Checkbox an, wenn Sie rückwärts paginieren/foliieren wollen. Dann klicken Sie auf Paginieren.
